# Amadeo-Michi--Web-cam-Calculator-MIDI
Algorithmic Music MIDI File Created by Detection of Hand by Web-cam and a Summatory in a Calculator

Try running Thonny - use your fingers on the webcam for 10 seconds to open the calculator, perform a sum in the calculator, and open the MIDI in MuseScore.
